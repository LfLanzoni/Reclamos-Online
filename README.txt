Desarrolador de aplicaciones Android

Aplicación “Reclamos Online” que permite a un usuario registrar reclamos a un municipio por diversas
situaciones (veredas rotas, baches, basura no recolectada, semáforos que no funcionan, etc).
Esta aplicación permite que el usuario ingrese el punto en el mapa donde se registra el reclamo, le adjunte una
foto, y una descripción.
También la aplicación permitirá
- buscar reclamos cercanos a una posición actual en el mapa.
- Mostrar en el mapa todos los reclamos de un mismo tipo.
- Generar un itinerario para resolver N itinerarios del mismo tipo.

No posee clave de API
